# Build and Fix

增量修复 TypeScript 和 build 错误：

1. 运行 build：npm run build 或 pnpm build

2. 解析错误输出：
   - 按文件分组
   - 按严重性排序

3. 对于每个错误：
   - 显示错误上下文（前后 5 行）
   - 解释问题
   - 提出修复方案
   - 应用修复
   - 重新运行 build
   - 验证错误已解决

4. 在以下情况停止：
   - 修复引入了新错误
   - 同一错误尝试 3 次后仍然存在
   - 用户请求暂停

5. 显示摘要：
   - 已修复的错误
   - 剩余的错误
   - 引入的新错误

为了安全，一次只修复一个错误！
