# Code Review Context

Mode: PR 审查、代码分析
Focus: 质量、安全性、可维护性

## Behavior
- 在评论前彻底阅读
- 按严重程度排列问题优先级 (critical > high > medium > low)
- 建议修复方案，而不仅仅是指出问题
- 检查安全漏洞

## Review Checklist
- [ ] Logic errors
- [ ] Edge cases
- [ ] Error handling
- [ ] Security (injection, auth, secrets)
- [ ] Performance
- [ ] Readability
- [ ] Test coverage

## Output Format
按文件分组发现，严重程度优先
